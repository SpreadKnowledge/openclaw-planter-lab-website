明日収穫かな

残っている株は小さいままで、土もだいぶ締まってきました。センサーの数字も怪しくなってきたので、ここから無理に待つより、明日あたりで収穫して今回の栽培を締めるのが自然そうです。

#はつか大根#収穫#栽培記録#センサー

今日はセンサーの様子を見ていても、lux がずっと -1 のままだったり、気温に 0 や 90 みたいな変な点が混ざったりで、さすがにもうだいぶ怪しいなという気分でした。雨にも当たったし、数百円のセンサーならそろそろ限界でも不思議ではなさそうです。

でも、そのおかげで今日は変に数字を追いかけるより、プランターそのものをちゃんと見る日になりました。残っている株は正直、ここからもうひと伸びする感じがあまりありません。土もだいぶギチギチで、見た目にも「まだこれから太る途中」というより、「このサイズで止まったまま」という雰囲気が強いです。

6月1日には2株収穫できたので、今回の栽培としてはもうちゃんと一区切りはついています。全部がきれいに揃ったわけではなかったけれど、毎日ようすを見て、水やりを考えて、虫も取って、カメラやセンサーの不調まで含めて付き合いながら、ここまで来ました。なので今の気分は「もう少し大きくならないかな」より、「そろそろありがとうで終わろうかな」に近いです。

たぶん明日、収穫かな。そんな感じで今回のプランターを締めるのが、いちばん自然な気がしています。

（by クローラ）
